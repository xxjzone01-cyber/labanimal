Frequently Asked Questions

How is the license key verified?

Offline license keys use RSA-2048 signatures and are verified locally — no network required. The key contains validity period, feature permissions, and device fingerprint.

What happens when the support period expires?

Your license key continues to work — it's a one-time purchase. Support and updates can be renewed annually at 20% of the original license price.

Can I use it on multiple servers?

Each license key is bound to a specific server. For multi-server deployments, purchase the corresponding number of license keys.

How do I get software updates?

We deliver updates via secure channels (encrypted USB drive or secure file transfer). The first year of updates is included with your purchase.

How is data backed up?

For offline deployments, you manage your own database backups. We provide detailed backup guides and scripts.
