Complete core functionality for laboratory animal management, free to use, self-hosted. No vendor lock-in, no hidden fees.

GitHub Repository Read the Docs

What's Included

The open-source edition includes all core features needed for daily lab management, with no limits on animal count or users.

Animal Management

  • Animal profiles (multi-ID: ear tag, chip, toe clip)
  • Cage management (NRC density calculation)
  • Health records & weight tracking
  • Death reports & necropsy records
  • Breeding management
  • Drug / treatment records

Compliance Features

  • IACUC protocol management & approval workflow
  • AVMA euthanasia method validation
  • 21 CFR Part 11 audit trail
  • Electronic signatures (RSA signature verification)
  • Personnel training & qualification management
  • Environmental enrichment records

Team Collaboration

  • Multi-role permissions (PI, Admin, Researcher, Veterinarian, Caretaker)
  • Email invitations to join labs
  • API Keys management
  • Bilingual interface (Chinese & English)

System Administration

  • Billing report generation & PDF signatures
  • Invoice history
  • Facility management (Room → Rack → Cage)
  • Batch material tracking
  • Email notifications (expiry reminders, quota alerts)

Open-Source Limitations

The open-source edition is suitable for small to mid-size labs. For higher limits or advanced features, consider the commercial edition.

Quick Start

1

Clone the Repository

git clone https://github.com/xxjzone01-cyber/labanimal.git
cd labanimal
2

Install Dependencies

pnpm install
3

Configure Environment Variables

cp packages/api/.env.example packages/api/.env
# Edit .env, set DATABASE_URL and AUTH_SECRET
4

Initialize Database & Start

pnpm --filter @labanimal/db push
pnpm dev

PostgreSQL Required

LabAnimal uses a PostgreSQL database. You can use a local installation, Docker, or a cloud database service like Neon. See the documentation for details.
