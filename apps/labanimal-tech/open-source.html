包含功能

开源版包含日常实验室管理所需的全部核心功能，不限制动物数量或用户数。

动物管理

  • 动物档案（多标识：耳标、芯片、趾 clip）
  • 笼位管理（NRC 密度计算）
  • 健康记录与体重追踪
  • 死亡报告与尸检记录
  • 繁殖管理
  • 药物/治疗记录

合规功能

  • IACUC 协议管理与审批流程
  • AVMA 安乐死方法验证
  • 21 CFR Part 11 审计追踪
  • 电子签名（RSA 签名验证）
  • 人员培训与资质管理
  • 环境富化记录

团队协作

  • 多角色权限（PI、管理员、研究员、兽医、饲养员）
  • 邮件邀请加入实验室
  • API Keys 管理
  • 中英文双语界面

系统管理

  • 计费报告生成与 PDF 签名
  • 发票历史记录
  • 设施管理（房间→机架→笼位）
  • 批次物料追踪
  • 邮件通知（到期提醒、配额告警）

快速开始

1

克隆仓库

git clone https://github.com/xxjzone01-cyber/labanimal.git
cd labanimal
2

安装依赖

pnpm install
3

配置环境变量

cp packages/api/.env.example packages/api/.env
# 编辑 .env，设置 DATABASE_URL 和 AUTH_SECRET
4

初始化数据库并启动

pnpm --filter @labanimal/db push
pnpm dev

需要 PostgreSQL

LabAnimal 使用 PostgreSQL 数据库。您可以使用本地安装、Docker 或 Neon 等云数据库服务。详见 文档。

立即开始使用

5 分钟内完成部署，开始管理您的实验动物设施。
